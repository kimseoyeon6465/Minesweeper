프로젝트명: Minesweeper

프로젝트 개요
2차원 데이터 구조와 사용자 입력을 기반으로 상태 변화를 처리하는
콘솔 환경의 지뢰 탐색 프로그램입니다.

주요 기능
- 2차원 배열 기반 맵 데이터 생성
- 사용자 입력 이벤트 처리
- DFS 알고리즘을 활용한 인접 영역 탐색
- 상태 관리 및 종료 조건 판별

담당 역할
- 전체 로직 설계 및 구현 (1인 개발)
- DFS 탐색 로직 구현 및 상태 관리
- WinAPI 기반 입력 처리 및 좌표 변환

기술 스택
C++ / WinAPI / Console Application

개발 목적 및 배운 점
DFS 재귀 구조를 실제 동작하는 시스템에 적용하며,
상태 관리와 종료 조건 설계의 중요성을 경험했습니다.
